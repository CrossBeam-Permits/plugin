# LTIA Demolition -- Plan Submittal Requirements

Source: 2025 California Building Code (Title 24, Part 2), Section 3303.1; 2025 California Existing Building Code (Title 24, Part 10), Sections 106 and 601.2; Standard industry practice for LTIA demolition permits.

---

## Required Plan Sheets

### A. Demolition Floor Plan

The demolition floor plan is the core document. It must show:

1. **Existing floor plan** of the tenant space showing all rooms, doors, and demising walls between adjacent tenants.
2. **Demolition scope** clearly indicated -- items to be removed shown with dashed lines or shading per standard demolition graphic conventions.
3. **Items to remain** clearly distinguished from items to be demolished.
4. **Area and occupancy type** of each room listed.
5. **Area of interior demolition** in square feet.

### B. Site Plan

A dimensioned or scaled drawing that includes:

1. Property lines with dimensions (lot width x depth)
2. All buildings on the property
3. All easements and setbacks from property lines and distances between structures
4. Driveways
5. Streets with street names and sidewalks adjacent to the property
6. North arrow orientation
7. Fence locations with heights and type

### C. Path of Travel / Accessibility Plan

1. Path of travel showing maintained compliance with disabled access requirements
2. Main entrance location
3. Path of travel from public sidewalk/parking to the tenant space
4. Restrooms, drinking fountains, and telephones (if provided)

### D. Reflected Ceiling Plan

Required if there is demolition scope in the ceiling area. Must show:
1. Existing ceiling grid and tiles to be removed
2. Light fixtures to be removed
3. Any ceiling-mounted devices (speakers, sprinkler heads, smoke detectors) -- note which remain and which are removed
4. Sprinkler heads must remain functional or be properly relocated/capped

### E. Roof Plans (If Applicable)

If work scope includes areas on the roof, fully dimensioned roof plans are required.

### F. Egress Plan

1. Egress path from the tenant space to building exits
2. All exit locations clearly marked
3. Fire separation walls, fire-rated corridors, door ratings and door locations

## Required Notes on Plans

The following notes shall appear on the demolition plans:

1. **General compliance**: "All materials, equipment, installation and work shall comply with the latest edition of the California Building Code (CBC), California Electrical Code (CEC), California Energy Code (T24), California Green Code (CALGreen), California Fire Code (CFC), California Mechanical Code (CMC), California Plumbing Code (CPC), California Residential Code (CRC), and [City] Municipal Code."

2. **Fire/life safety/accessibility**: "All applicable fire, life, safety, and accessibility requirements shall be maintained."

3. **Utility capping**: "All utility lines must be properly and safely capped as needed."

4. **Shoring**: "Contractor is responsible for all temporary shoring prior to the start of demolition."

5. **Waste diversion**: "65% minimum of nonhazardous construction and demolition waste shall be recycled and/or salvaged for reuse." (CALGreen 5.408.1)

6. **Scope limitations**: Permit is for interior, non-structural demolition only. No other improvements are allowed. All tenant improvements shall be processed under a separate permit.

## Required Documentation (Beyond Plans)

| Document | When Required |
|----------|--------------|
| **Hazardous material inventory** | If chemicals stored on site -- list type and quantities |
| **AQMD notification number** | Required for asbestos before demolition permit is issued |
| **Lead testing results** | For buildings constructed 1978 or earlier |
| **Waste hauler receipts** | Required for final inspection sign-off |
| **Property owner information** | Name, telephone, email |
| **Contractor information** | Name, license, telephone, email |

---

## Common Plan Deficiencies (Items Frequently Missing)

1. Demising walls not clearly shown or labeled
2. Fire-rated walls not identified with ratings
3. Egress path not shown
4. Path of travel / accessibility compliance not addressed
5. Reflected ceiling plan missing when ceiling work is in scope
6. Occupancy type not listed for each room
7. Required notes missing from plans
8. Area of demolition not stated in square feet
9. Fire separation walls and door ratings not shown
