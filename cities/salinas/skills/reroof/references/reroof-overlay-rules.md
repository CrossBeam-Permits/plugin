# Reroof Overlay and Tear-Off Rules

Source: 2025 California Building Code (Title 24, Part 2), Chapter 15, Section 1512; California Residential Code (Part 2.5), Section R908.

---

## Two Methods of Reroofing

### Roof Replacement (Tear-Off) — CBC 1512.2
Full removal of all existing roof covering materials down to the deck. Required when overlay conditions cannot be met.

**What must be removed:**
- All existing roof covering layers
- Deteriorated or water-soaked underlayment
- Damaged sheathing must be repaired or replaced

**Exceptions to full removal:**
1. Existing self-adhered ice barrier membrane may remain if sheathing is NOT water-soaked or deteriorated — cover with new ice barrier membrane
2. Existing self-adhered underlayment may remain if sheathing is adequate — cover with new underlayment per Table 1507.1.1
3. Two self-adhered underlayment layers permitted only if:
   - Second layer cannot be removed without damaging deck
   - Permitted by both roof covering and underlayment manufacturers
   - Sheathing is not water-soaked or deteriorated
   - Total buildup at walls, valleys, edges, and end laps ≤2 layers

### Roof Recover (Overlay) — CBC 1512.3 / CRC R908.3
Installing new roof covering over the existing roof without removing it.

---

## When Is Overlay PERMITTED?

Overlay is allowed when ALL of the following are true:

| Condition | Requirement |
|---|---|
| Existing layers | **≤1 existing layer** (new layer makes 2 total maximum) |
| Existing condition | NOT water-soaked or deteriorated |
| Existing material | NOT slate, clay tile, cement tile, or asbestos-cement tile |
| Installation | Per approved manufacturer instructions |

**Additional permitted recover scenarios (CBC 1512.3):**
1. Complete separate roofing system (e.g., standing-seam metal, PV panels) designed to transmit loads directly to the structure
2. Metal panel, metal shingle, or concrete/clay tile installed over existing wood shakes (with Section 1512.3.1 compliance — see below)
3. New protective coating applied over existing coating, metal panel, built-up, SPF, metal shingle, mineral roll, modified bitumen, or single-ply

---

## When Is Overlay PROHIBITED? (Must Tear Off)

| Condition | Why | Code |
|---|---|---|
| **2 or more existing layers** | Maximum 2 layers total — 3rd layer not permitted | CBC 1512.3; CRC R908.3.1 |
| **Existing is slate, clay, cement, or asbestos-cement tile** | Cannot overlay these materials | CBC 1512.3 |
| **Existing is water-soaked or deteriorated** | Deck integrity compromised | CBC 1512.3 |
| **Existing sheathing is damaged** | Must expose and repair deck | CBC 1512.2 |

---

## Special Case: Recover Over Wood Shakes/Shingles (CBC 1512.3.1)

When installing metal panel, metal shingle, or concrete/clay tile OVER existing wood shakes or shingles:
- If the installation creates a **combustible concealed space** between old and new covering
- The **entire roof surface** must be covered with:
  - Gypsum board
  - Mineral fiber
  - Glass fiber
  - Other approved noncombustible material

This prevents fire from spreading through the concealed space between the old wood and new covering.

---

## Reinstallation of Existing Materials (CBC 1512.4)

Existing materials may be reused in limited cases:
- **Slate, clay, cement tile:** May be reinstalled EXCEPT tiles that are damaged, cracked, or broken
- **Flashing and metal edgings:** May be reinstalled EXCEPT if rusted, damaged, or deteriorated

---

## Flashing Requirements on Reroof (CBC 1512.5)

- All flashings must be reconstructed per manufacturer installation instructions
- Metal flashing used with bituminous roofing materials: **prime before installation**
- Existing flashing may be reused only if not rusted, damaged, or deteriorated

---

## Sheathing Inspection (Tear-Off)

When tear-off exposes the roof deck, the inspector should verify:
1. Sheathing is structurally sound (no rot, delamination, or insect damage)
2. Fasteners are adequate (nails not backing out)
3. Gaps between sheathing panels are within tolerance
4. Any water-damaged sections are replaced

**This is a critical inspection point** — the sheathing inspection typically occurs after tear-off and before new underlayment/covering installation.

## Decision Flowchart

```
Is this a tear-off or overlay?
│
├─ TEAR-OFF (Roof Replacement)
│  → Remove all layers to deck (CBC 1512.2)
│  → Inspect sheathing
│  → Repair/replace damaged sheathing
│  → Install new underlayment + covering
│  → 65% waste diversion required (CALGreen 5.408)
│
└─ OVERLAY (Roof Recover)
   → How many existing layers?
   │
   ├─ 0 or 1 existing layer
   │  → Check: existing NOT slate/clay/cement/asbestos tile?
   │  → Check: existing NOT water-soaked/deteriorated?
   │  → Check: manufacturer approves overlay?
   │  → If all YES → Overlay permitted
   │  → Install new covering over existing
   │
   └─ 2 or more existing layers
      → OVERLAY PROHIBITED
      → Must tear off (CBC 1512.3)
```

## Common Review Findings

1. **Applicant checks "Reroof Over Existing" but existing already has 2 layers** — FAIL. Must tear off. Ask applicant to confirm number of existing layers.
2. **Overlay over wood shakes** — Technically allowed with noncombustible separation, but this is uncommon. FLAG for inspector review.
3. **Partial tear-off** — If only a portion of the roof is torn off, the overlay rules apply to the remaining portion. New and existing sections must integrate properly at transitions.
