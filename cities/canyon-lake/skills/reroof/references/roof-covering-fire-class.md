# Roof Covering Fire Classification

Source: 2025 California Building Code (Title 24, Part 2), Chapter 15, Section 1505; California Residential Code (Part 2.5), Section R902.

---

## Fire Classification Ratings

Roof coverings are tested per ASTM E108 or UL 790 and rated:

| Class | Fire Resistance | Common Materials |
|---|---|---|
| **A** | Highest — effective against severe fire exposure | Concrete/clay tile, metal, asphalt (fiberglass-reinforced), slate |
| **B** | Moderate — effective against moderate fire exposure | Some treated wood shakes |
| **C** | Minimum — effective against light fire exposure | Some roll roofing, treated wood shingles |

## Minimum Classification by Construction Type (CBC Table 1505.1)

| Construction Type | Minimum Roof Class |
|---|---|
| Type IA, IB, IIA, IV, VA | Class B |
| Type IIB, IIIB, VB | Class C |

**Exception:** Nonclassified roofing is permitted on Group U (utility) buildings with ≥6 feet fire-separation distance from adjacent structures.

## California 50% Replacement Rule (CBC 1505.1.2 / CRC R902.1.2)

This is California-specific and critical for reroof permits:

> Where more than 50% of the total roof area is replaced within any one-year period, OR where any roof covering alteration, repair, or replacement is performed, the **entire roof covering must be fire-retardant rated at minimum Class C.**

This means:
- Any reroof project (even partial) triggers the fire-rating requirement
- The new covering must be listed/labeled with its fire classification
- Materials without a fire classification rating cannot be used on reroof projects in California

## Fire-Retardant Treated Wood (CBC 1505.6)

Wood shakes and shingles used as roof covering must:
- Comply with ICC-ES Evaluation Guidance EG107
- Pass a 10-year natural weathering test (minimum 5 years completed at time of listing)
- Be approved by the California State Fire Marshal before sale or application
- Meet ASTM D2898 for fire-retardant treatment

## Very High Fire Hazard Severity Zones (VHFHSZ)

In areas designated VHFHSZ by CAL FIRE:
- **Class A roof covering is mandatory** (not just Class C)
- Applies regardless of construction type
- Check local fire authority for zone maps
- Many Southern California communities are partially or fully in VHFHSZ

## Product Listing / Labeling

All roof coverings must bear a label from an approved testing agency showing:
- Manufacturer name
- Fire classification (A, B, or C)
- Test standard (ASTM E108 or UL 790)
- Listing/identification number

## Practical Review Notes

1. **Asphalt shingles** — Modern fiberglass-reinforced shingles are Class A. Organic-mat shingles (largely discontinued) were Class C.
2. **Concrete and clay tile** — Inherently Class A. No fire concerns.
3. **Metal** — Inherently noncombustible; Class A when installed over proper underlayment.
4. **Wood shakes/shingles** — Only fire-retardant treated versions are permitted in California. Untreated wood roofing is effectively prohibited statewide.
5. **Built-up, modified bitumen, single-ply** — Classified as assemblies (covering + underlayment + deck). Must be tested as complete system.
