## Plan-Check Checklist (applies to both modes)

Every item here is verified at plan-check time. Severity drives reviewer-mode output.

| # | Check | Severity | What to verify | Code |
|---|---|---|---|---|
| 1 | **Service capacity adequate?** | ❌ **fail** | If full rewire: verify panel can handle all circuits; load calc per 220.83(A) | CEC 220.83(A) |
| 2 | **Outlet spacing correct?** | ❌ **fail** | Habitable rooms: no point >6' from receptacle; kitchen countertop: no point >24" from receptacle | CEC 210.52 |
| 3 | **GFCI protection?** | ❌ **fail** | Required in: bathrooms, kitchens, garages, outdoors, laundry, crawl spaces, basements, near sinks/tubs | CEC 210.8(A) |
| 4 | **AFCI protection?** | ❌ **fail** | Required in: kitchens, bedrooms, family/living/dining rooms, hallways, closets, laundry, rec rooms | CEC 210.12(B) |
| 5 | **Dedicated circuits?** | ❌ **fail** | Kitchen: 2× 20A small-appliance; bathroom: 20A; laundry: 20A; garage: 20A | CEC 210.11(C) |
| 6 | **Wire type and sizing?** | ❌ **fail** | NM cable (Romex) for residential; #14 for 15A, #12 for 20A; conduit where exposed | CEC 334, Table 310.16 |
| 8 | **Contractor licensing?** | ⚠️ **flag** | Licensed C-10 Electrical Contractor required (CA B&P Code §7028). Owner-builder exemption per B&P §7044 only for owner-occupied residential where owner personally performs the work. Verify active CSLB license and workers' comp insurance. | CA B&P §7028, §7044 |
| 1 | **Copper wire mandate** `[DPMC]` | ⚠️ **flag** | Dana Point municipal code (Ch. 8.14) requires copper conductors for all wiring No. 6 AWG and smaller; this is more restrictive than NEC which permits aluminum; all branch circuit home runs must be copper | CRITICAL |
| 2 | **Aluminum feeder process** `[DPMC]` | ⚠️ **flag** | To use aluminum for service entrance or feeder runs, applicant must obtain written Building Official approval BEFORE permit issuance; include approval letter in permit application package | CRITICAL |
| 3 | **SDG&E work order** `[DPMC]` | ⚠️ **flag** | If any service/panel work is involved in the rewire scope, obtain SDG&E work order number and provide to Building Division before final inspection | Required |
| 4 | **No aluminum branch circuits** `[DPMC]` | ⚠️ **flag** | Regardless of wire gauge, no aluminum for branch circuit wiring — applies to all circuits, not just those ≤ No. 6 AWG | CRITICAL |
| 5 | **Permit expiration** `[DPMC]` | ⚠️ **flag** | 180 days from issuance; double permit fee if work commenced without permit | Required |
| 6 | **E360 Portal** `[DPMC]` | ⚠️ **flag** | Submit via E360 Digital Portal; track permit status via eTRAKiT | Process |
| 7 | **Coastal Zone** `[DPMC]` | ⚠️ **flag** | Typically CDP-exempt for internal rewire of existing structure; confirm with Planning if any exterior work is in scope | Conditional |
