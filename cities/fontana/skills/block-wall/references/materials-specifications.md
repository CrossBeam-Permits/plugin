# Block Wall Material Specifications

Source: 2025 California Residential Code (Title 24, Part 2.5), Section R606.2; California Building Code (Part 2), Section 2103; TMS 602 Specification for Masonry Structures.

---

## Concrete Masonry Units (CMU)

### Standard Requirement
All load-bearing concrete masonry units must conform to **ASTM C90**.

| Property | Requirement | Standard / Code |
|---|---|---|
| Standard | ASTM C90 | CRC R606.2.1; CBC 2103.1 |
| Grade | **Grade N** (moisture-controlled) for exterior use | ASTM C90 |
| Minimum f'm | **1,500 psi** (specified compressive strength of masonry) | TMS 402 / CRC R606 |
| Weight class | Normal weight preferred (for mass and stability) | Lightweight permitted with engineering |
| Nominal size | 8" x 8" x 16" (standard) | 7-5/8" x 7-5/8" x 15-5/8" actual |
| Bond pattern | **Running bond required** | CRC R606.3.7.1 |

### Block Types

| Block Type | Description | Use |
|---|---|---|
| **Standard hollow CMU** (2-cell) | Most common; two hollow cells separated by web | Standard freestanding walls |
| **Solid CMU** | No cells or cells filled at factory | Additional mass/impact resistance |
| **Split-face CMU** | Rough textured face, standard structural properties | Decorative property line walls |
| **Slump block** | Irregular adobe-like appearance | Decorative; verify f'm meets 1,500 psi |
| **Scored/fluted CMU** | Vertical grooves on face | Decorative; same structural specs |
| **Bond beam block** | U-shaped (open top) for horizontal rebar | Required at top course and intermediate bond beams |
| **Pilaster block** | Wider units (16"x16" or 16"x24") for columns | Pilasters at ends, corners, or intermediate support |

### Grade N vs. Grade S
- **Grade N (moisture-controlled):** Required for exterior use and all below-grade applications. Controls moisture content at time of delivery.
- **Grade S:** General use, no moisture-content limitation. Acceptable for interior partitions only.

For freestanding exterior block walls, **always Grade N**.

## Mortar

### Types for California Seismic Zones

Per CRC R606.2.8.3, mortar for masonry in **Seismic Design Categories D0, D1, and D2** (most of California) must be **Type M or S Portland cement-lime or mortar cement**.

| Mortar Type | Compressive Strength | Acceptable? | Best Use |
|---|---|---|---|
| **Type M** | 2,500 psi | **YES** | Below-grade, ground contact, high-wind areas, highest durability |
| **Type S** | 1,800 psi | **YES** | Above-grade freestanding walls |
| Type N | 750 psi | **NO — not for freestanding exterior walls in SDC D** | Interior partitions only |
| Type O | 350 psi | **NO** | Not suitable for structural masonry |

### Proportion Specifications (ASTM C270)

| Mortar Type | Portland Cement | Hydrated Lime | Sand |
|---|---|---|---|
| **Type M** | 1 part | 1/4 part | 3-3/4 parts |
| **Type S** | 1 part | 1/2 part | 4-1/2 parts |

Pre-mixed bagged mortar conforming to ASTM C270 (Type M or S) is acceptable and commonly used.

### Mortar Joint Requirements (CRC R606.3.1)
- Bed and head joints: **3/8 inch** nominal thickness
- Starting course bed joint over foundation: 1/4" to 3/4" range
- Full head and bed joints for solid masonry (CRC R606.3.2.1)
- Hollow units: head and bed joints filled solidly for a distance from face not less than face shell thickness (CRC R606.3.2.2)

## Grout

### Standard Requirements

| Property | Requirement | Code / Standard |
|---|---|---|
| Standard | **ASTM C476** | CRC R606.2.12; CBC 2103.3 |
| Minimum f'c | **2,000 psi** | ASTM C476 |
| Slump | **8-11 inches** (pourable consistency) | TMS 602 |
| Fine grout | For cells <=2" wide | CRC Table R606.3.5.1 |
| Coarse grout | For cells >2" wide | CRC Table R606.3.5.1 |

### Grout Proportions (CRC Table R606.2.12)

| Type | Portland Cement | Hydrated Lime | Fine Aggregate | Coarse Aggregate |
|---|---|---|---|---|
| **Fine** | 1 | 0 to 1/10 | 2-1/4 to 3x cement volume | — |
| **Coarse** | 1 | 0 to 1/10 | 2-1/4 to 3x cement volume | 1 to 2x cement volume |

### Alternative
Type M or Type S mortar thinned to pouring consistency (with additional water) is permitted as grout per CRC R606.2.12.

### Grouting Rules
- **All cells containing reinforcement must be fully grouted** — no exceptions
- Non-reinforced cells: grouting optional but recommended for durability
- Grout must completely surround reinforcing bars and fill the cell
- Consolidation required: mechanical vibration (preferred) or puddling with a rod
- Reconsolidation: 3-5 minutes after initial pour to account for settlement

## Reinforcing Steel

### Standard Requirements

| Property | Requirement | Standard |
|---|---|---|
| Standard | **ASTM A615** (carbon steel) or **ASTM A706** (low-alloy, weldable) | TMS 602 Article 2.4 |
| Grade | **Grade 60** (60,000 psi yield strength) | Standard for structural masonry |
| Minimum bar size | **#4** (1/2" diameter, 0.20 sq in area) | TMS 402 for structural walls |
| Cover in grouted cells | **5/8" minimum** | TMS 402 |
| Cover in mortar joints (joint reinforcement) | **5/8" minimum** from exposed face | CRC R606.3.4 |
| Cover exposed to weather or soil | **2 inches minimum** | CRC R606.3.4 |

### Lap Splice Requirements

| Bar Size | Diameter | Minimum Lap (48 db) | Notes |
|---|---|---|---|
| **#4** | 0.500" | **24"** | Standard for walls <=6 ft |
| **#5** | 0.625" | **30"** | Walls >6 ft or per engineering |
| **#6** | 0.750" | **36"** | Engineered walls / pilasters |

Per CBC 2107.2.1, lap splice length shall not be less than 40 bar diameters (alternative to TMS 402 Section 6.1.7.1). The more conservative 48 db from TMS 402 is commonly used in practice.

Where design tensile stresses exceed 80% of allowable, increase lap by 50% (but need not exceed 72 db).

### Rebar Positioning
- Vertical rebar must be **centered** in the grouted cell
- Use positioning devices (spacers, wire chairs) to hold rebar in place during grouting
- Rebar must not contact block face shells — minimum 1/4" clearance between rebar and masonry unit (CRC R606.3.5.3)
- At splices, bars must overlap for the full lap length with grout between them

### Corrosion Protection (CRC Table R606.3.4.1)
- Interior walls: ASTM A641, Class 1 (standard galvanized)
- Exterior walls embedded in mortar/grout: ASTM A641, Class 3
- Exterior walls not fully embedded: ASTM A153, Class B-2 (hot-dip galvanized)
- Joint reinforcement in exterior walls: ASTM A153, Class B-2
- Stainless steel: ASTM A167, Type 304 (for any exposure)

## Concrete (for Footings)

| Property | Requirement |
|---|---|
| Minimum f'c | **2,500 psi** (3,000 psi typical) |
| Slump | 4-6 inches (standard placement) |
| Maximum aggregate size | 3/4" to 1" |
| Curing | Minimum 24 hours before starting masonry; 3 days preferred |

### Expansive Soil Considerations
Cities with known expansive soils (e.g., Laguna Niguel — expansive soils city-wide) may require:
- **Type V cement** (sulfate-resistant) — minimum 4,500 psi concrete
- Deeper footings (24"+ below grade)
- Soils report before construction

## Second-Hand Units (CRC R606.2.7 / CBC 2103.1.1)

Second-hand (salvaged) masonry units may be reused ONLY if they:
- Conform to the requirements of new units (ASTM C90)
- Are whole and sound, free from cracks and defects
- Have all old mortar cleaned from the unit before reuse
