# Tile Weight and Structural Adequacy

Source: 2025 California Building Code (Title 24, Part 2), Sections 1607.13, 1507.3; California Residential Code (Part 2.5), Sections R301.1, R905.3.

---

## Why Tile Weight Matters

Clay and concrete tile are among the heaviest roof coverings. When a reroof project switches from a lighter material (like asphalt shingles) to tile, the existing roof framing may not be adequate to carry the additional dead load.

### Typical Roof Covering Weights

| Material | Approximate Weight (psf) |
|---|---|
| Asphalt/composition shingles | 2–4 |
| Wood shingles/shakes | 3–4 |
| Metal panels | 1–2 |
| Concrete tile | **9–12** |
| Clay tile | **8–15** |
| Slate | 7–10 |
| Built-up roofing (4-ply) | 3–5 |
| Single-ply membrane | 0.5–1.5 |

Tile can be **3–7 times heavier** than the asphalt shingles it often replaces.

---

## When Structural Analysis Is Required

Structural analysis or engineering review should be triggered when:

1. **Weight increase exceeds 3 psf** over the original roofing material — this is a practical threshold used by many jurisdictions
2. **Original framing was not designed for tile** — homes built with composition roofing may have lighter rafters/trusses
3. **Roof spans are long** (>16 feet unsupported) — more sensitive to added dead load
4. **Existing framing shows signs of distress** — sagging ridges, deflected rafters, cracked members
5. **The building is in a high seismic zone** — additional mass increases seismic forces on the structure

### Code Basis
- **CBC 1607.13:** Roof loads must account for dead load (weight of materials) + live load (workers, equipment during construction) + environmental loads (wind, seismic)
- **CRC R301.1:** Requires buildings to be designed for loads per accepted engineering practice
- The code does not specify a psf threshold for "when to require analysis" — this is typically a jurisdiction-level decision

## ICC-ES Evaluation Reports (ICC-ESR)

### What It Is
An ICC-ES Evaluation Report (ESR) is an independent third-party assessment that a roofing product meets code requirements. The ESR number is a unique identifier (format: ESR-XXXX).

### Why It Matters for Tile
- Verifies the tile product complies with ASTM C1167 (clay) or ASTM C1492 (concrete)
- Confirms fire classification rating
- Documents tested weight, dimensions, and installation requirements
- Provides wind resistance data

### Review Guidance
- **If ICC-ESR number is provided:** Note it in the review. Validation of the ESR against the ICC-ES database is city staff's responsibility, not the AI reviewer's.
- **If ICC-ESR number is NOT provided for tile:** FLAG — request the applicant provide the ESR number or equivalent product approval documentation.
- **ICC-ES reports can be looked up at:** evaluate.icc-es.org (for staff reference)

## Practical Review Checklist for Tile

| Check | What to Verify | Action if Missing |
|---|---|---|
| Tile weight (psf) | Must be stated on application | FLAG — request weight |
| Previous roof material | What was on the roof before? | Compare weights |
| Weight difference | Calculate: new weight − old weight | If >3 psf increase, FLAG for structural review |
| ICC-ESR number | Product approval documentation | FLAG if tile but no ESR |
| Deck type | Solid sheathing required for tile (CBC 1507.3) | FAIL if spaced sheathing (unless Seismic A/B/C) |

## Tile-Specific Installation Requirements

### Underlayment (CBC 1507.3 / CRC R905.3)
- ASTM D226 Type II, D2626, or D6380 Class M
- Double underlayment required for slopes 2.5:12 to 4:12

### Fastening (CBC Table 1507.3.7)
- 1–2 fasteners per tile based on wind speed and building height
- Minimum 11-gage wire, 5/16" head
- Penetration: 3/4" into deck or through sheathing (whichever less)

### Valley Flashing
- Extend 11" from centerline on each side
- Splash diverter ≥1" high

### Deck
- Solid sheathing required
- Exception: Spaced lumber (1"×4") permitted only in Seismic Design Categories A, B, C
